```markdown
# 🌿 EcoMarket — Sustainable Products Store

> Built with pure HTML5 & CSS3. No JavaScript. No frameworks. Just clean code.

## 🔗 Live Demo
[Coming soon — deploy in progress]

## 🛠️ Tech Stack
HTML5 Semantic  →  Structure
CSS3 Pure       →  Styling & Animations
Google Fonts    →  Merriweather + Lato
SVG Icons       →  Inline, no libraries
Git & GitFlow   →  Version control

## 📁 Project Structure
ECOMARKET_FANDINONICOLAS/
├── css/
│   ├── main.css         # Reset, variables, typography
│   ├── layout.css       # Header, Footer, Grid systems
│   ├── components.css   # Cards, Buttons, Inputs
│   └── animations.css   # Keyframes and transitions
├── img/
│   ├── products/        # Product images
│   ├── icons/           # Eco badge icons
│   └── hero/            # Hero and banner images
├── views/
│   ├── productos.html   # Store page
│   ├── sobre-nosotros.html  # Our Story page
│   └── contacto.html    # Contact page
├── index.html           # Home page
└── README.md
```

## 🌱 Pages
/index.html → Home page.

/views/articulo.html, articulo2.html, articulo3.html, articulo4.html → Independent article views featuring a 2-column CSS Grid layout (70% main content / 30% sticky sidebar), breadcrumbs, drop caps, pull quotes, and related posts.

/views/productos.html → Store catalog.

/views/sobre-nosotros.html → Our Story.

/views/contacto.html → Contact form and details.
